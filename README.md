## Cover Palette

Base color: `#2ba283`

Palette for generated post covers:

- Teal: `#2ba283` - spirit / nature / presence
- Blue: `#2c4fa0` - wisdom / depth / word
- Violet: `#7d2ca0` - authority / mystery / foundation
- Crimson: `#a02c4f` - love / warmth / family

These four colors are built as a harmonious set from the base teal with similar saturation and lightness, shifting only the hue.
